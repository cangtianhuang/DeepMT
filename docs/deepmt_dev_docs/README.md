# DeepMT 开发阶段文档（重构版）

本组文档用于替换旧版“开发状态”说明，目标不是记录零散进度，而是直接指导后续编码智能体完成一个同时满足以下三类目标的项目：

1. **研究目标**：能够形成论文第六章所需的实验闭环，真实发现框架问题，而不是只做概念验证；
2. **展示目标**：能够稳定演示“数据采集 → MR 生成 → 批量测试 → 缺陷分析 → 报告输出”的完整链路；
3. **工程目标**：具备插件化、可复现、可扩展、可持续运行的基本生产属性。

## 文档使用方式

- 先读 `00_开发总览与执行原则.md`，统一目标与边界；
- 再按阶段顺序执行 `01` 至 `05`；
- 编码智能体在接任务前阅读 `06_编码智能体执行规范.md`，按其中的任务粒度推进；
- 旧版“Phase A：算子数据层完善”等内容以本组文档为准。
